OversizeAPI

A comprehensive API for Oversize applications that provides access to apps, features, ads, and other resources. This API uses consistent patterns and follows RESTful principles.

Pagination

List endpoints support pagination with page and limit query parameters.

apps

Application-related endpoints

List all applications

Retrieves a list of all applications

Authorizations:
ApiKeyAuth
query Parameters
platforms
Array of strings (PlatformType)
Items Enum: "iOS" "macOS" "tvOS" "watchOS" "visionOS"

Filter by platforms

status
string
Default: "released"
Enum: "released" "draft" "all"

Filter applications by release status

header Parameters
App-Version
string
Example: 1.2.3

Client application version

App-Bundle-ID
string
Example: com.oversize.App

Bundle identifier in reverse domain notation

Accept-Language
string
Example: en-US

Preferred language for response content

App-Store-ID
string
Example: 1234567890

Apple App Store application identifier
